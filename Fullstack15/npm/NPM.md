# NPM  ->  Node Package Manager

### Module  ->  a file that ends with .js extension
    - RU: модуль ->  файл, который заканчивается расширением .js 


### Install a library | package
```bash
npm install '...'
```


### Install package.json
```bash
npm init   =>  It will ask an information about the project
# RU: Он попросит информацию о проекте
```

```bash
npm init -y   =>  It will create a package.json file with default values
# RU: Он создаст файл package.json со значениями по умолчанию
```


### CND  ->  Content Delivery Network
    - RU: Сеть доставки контента
- `toastify-js`  ->  https://apvarun.github.io/toastify-js/
- `lodash`       ->  https://lodash.com/

 -  -  For using these libraries, we should put their cdn links in the `index.html` file.
 -  -  RU: Для использования этих библиотек, мы должны поместить их ссылки cdn в файл `index.html`.

### Semantic Versioning
ex:  1.12.124
1.  Major version  ->  главная версия
2.  Minor version  ->  второстепенная версия
3.  Bug fixes      ->  исправление ошибок
